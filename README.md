# NexCLI

## Project Overview

**NexCLI** is a modular command-line interface (CLI) application developed by **NexSemble**. It provides an interactive terminal experience to access utilities, generate UUIDs, a secure password generator, and more!

## Features (with commands to use in the '>>>' area)

- **Welcome Banner**: ASCII art welcome on startup
- **UUID Generator**: Generate unique identifiers (`uuid`)
- **Password Generator**: Generate 12-character strong passwords (`pwd`)
- **Info**: Display developer information (`info`)
- **Developer Tools**: Debug command for development (`dev_check1`)
- **Graceful Exit**: Clean exit command (`exit`)

## Getting Started

1. Run the application:
   ```bash
   python main.py
   ```
2. Type `run` when prompted to start the CLI
3. Type `help` or any command in the `>>>` prompt

## Project Structure

```
.
├── main.py     # Entry point with CLI loop and command handling
├── NexCLI.py   # Core Nexcli class with utility methods
└── README.md   # Project documentation
```

## Commands Available

| Command     | Description                      |
|-------------|----------------------------------|
| `uuid`      | Generate a random UUID           |
| `pwd`       | Generate a 12-character password |
| `info`      | Show project & developer info    |         
| `exit`      | Exit the application             |
